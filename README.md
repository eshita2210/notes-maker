
# <h2>Notes Maker Application</h2>

## Notes Maker

**_A simple notes generator application that is used to make notes, save, delete them._**

<img width="1100" alt="SS PROJECT" src="src/assets/img/SS PROJECT.png">

### Developed a dynamic notes generator application utilizing
- HTML
- CSS
- JavaScript

### Key Features
- Note creation
- Deletion
- Saving functionalities with enforced word limits
- A toggle mode for enhanced user experience
